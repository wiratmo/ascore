# Pemecahan kalimat menjadi perkata
Bagaimana memecah kalimat menjadi perkata kalau terdapat kata yang bersambung, eg. makan2, 240/. 
```
prediksi solusi :
- membedakan antara 
	- huruf
	- angka
	- karakter
	240/ 120 x 60 = 120 kpm, jadi kecepatan dani membaca adalah 120 kpm

```
